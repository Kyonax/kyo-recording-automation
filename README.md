# kyo-recording-automation

OBS automation toolkit for **[@kyonax_on_tech](https://github.com/kyonax)** content creation — overlays, scene configs, and scripts to get it right at capture time and skip the editing.

```
kyo-recording-automation/
  assets/       <- reference images
  overlays/     <- HTML/CSS/JS browser source overlays
  scripts/      <- OBS WebSocket automation
  config/       <- scene collections & profiles
```
